Fog Computing Conferences and Journals

Where can you find fog computing and edge computing papers?


Here is a list of international conferences, journals, and magazines that actively publish (or will publish) technical contents related for fog computing, edge computing, mobile edge computing, cloudlet, etc.
This page is compiled by Ashkan Yousefpour. Please contact us if you think a conference, journal, or magazine should be added to the list, or if there is any major conference, journal, or magazine that is left out.
